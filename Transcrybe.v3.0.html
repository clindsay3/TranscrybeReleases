Transcrybe 3.0

Transcrybe 3.0 updates the foundation of the app with a ground-up interface rebuild, making settings, windows, and controls more consistent as the app grows. It also includes a ton of the most-requested features: Major Features * Rebuilt Foundation - Behind the scenes, Transcrybe's app interface has been rebuilt from the ground up so future features can grow on a more consistent, reliable foundation. * Reduced Distractions - The subtitle window's controls fade away when you move the mouse out of the way, leaving just your subtitles hovering over the content. * Keyboard Shortcuts - A new shortcut makes it quick to start or stop subtitles from anywhere on your Mac. * Move to This Screen - Quickly places Transcrybe on top of whatever you're doing, even if it's fullscreen. * Custom Vocabulary - You can now provide words ahead of time that transcription might mistake, helping it transcribe them correctly in Fluid-supported languages. * Improved Accuracy - German and Italian now use NVIDIA Parakeet for faster and more accurate transcription. Chinese (Mandarin) now uses Speech Analyzer on macOS 26. Other Improvements * You can now change languages directly in the subtitle window by clicking the language name. * The Settings screen has been completely redesigned. * The History screen has been integrated into Settings. * Transcrybe has a new app icon. * Settings now includes copyable app diagnostics to make support requests easier. Bug Fixes * Opening the app while it is already running now opens the app's menu when no app window is visible, while respecting existing windows. * If translation languages need to be downloaded, Transcrybe now gives clearer recovery options. * If the app's menu bar icon is hidden in System Settings > Menu Bar, the notification now explains that. * Menu bar warnings are quieter when menu bar managers are running. * Subtitle and rolling caption placement is more consistent and stable. * Tons of minor bug fixes throughout the app.
