Transcrybe 3.0

* Fixed an issue where translation could stop at unexpected times, especially on Intel Macs. * New About screen with version, build number, and app stream details. * Fixed an issue where Simplified and Traditional Chinese labels could appear in the wrong language in the translated language menu. * Fixed onboarding so the purchase step no longer says “Purchase Complete” while you're still in the free trial. * Existing test licenses will no longer work properly after this update; create a new test license before using one for validation. * Fixes an issue where long sessions using built-in speech recognition could slow down as the current caption kept growing instead of clearing after completed text moved into history.
